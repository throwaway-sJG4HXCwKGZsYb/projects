# LINK Mobility task

Projects and tasks

## System requirements

- PHP 8.3
- Composer 2
- NPM

## Installation


1. Clone the project
2. Copy .env into .env.local
3. Setup these .env.local variables:

```
DATABASE_URL
JWT_PASSPHRASE
```
4. Run in the root directory:
```
composer install

npm install
```

5. Setup virtual hosts or run symfony server to start the project
